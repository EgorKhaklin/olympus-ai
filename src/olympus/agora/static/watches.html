Manage via CLI

Watches are operator-declared; the substrate doesn't auto-create them. Add via CLI:

$ invoke argos watch add <id> <path> [--glob "*.md"] [--action alert|errand:<name>]
$ invoke argos watch remove <id>
$ invoke argos scan          # force a scan now
$ invoke argos watches       # list (this page mirrors)

Errand whitelist for action: errand:... — today, session, recall, doctor. Anything else is refused at config-load.
