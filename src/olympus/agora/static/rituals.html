Grammar

daily HH:MM            every day at HH:MM
weekday HH:MM          Mon-Fri at HH:MM
weekend HH:MM          Sat-Sun at HH:MM
<day> HH:MM            monday/tuesday/.../sunday at HH:MM
monthly <N> [HH:MM]    1..28 of each month
every <N>m | every <N>h     every N minutes/hours
hourly                 on the hour

Manage via CLI

$ invoke chronos ritual add <id> <when> <do>
$ invoke chronos ritual remove <id>
$ invoke chronos tick                       # force evaluation now
$ invoke chronos check "<when-expr>"        # parse + project next 3 due

Errand whitelist: today, session, recall, doctor, ferry, spend, heal, blessing, speak — anything else is refused.

Per Chronos arc · daemon tick fires matching rituals; each → chronos.fired
