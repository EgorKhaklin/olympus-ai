## Safety boundaries

- **Vault page never returns secret values** â€” `/vault` returns location + sha256-prefix + bytes_known only
- **Mutations stay CLI-only**:
  - `--acknowledge-budget` (Plutus)
  - `vault deposit/forget/migrate` (Hades)
  - `chronos ritual add/remove` (Chronos)
  - `argos watch add/remove` (Argos)
  - `hephaestus apply` (Hephaestus-PR)
- **The single new POST (`/library/ingest`)** has bounded effect â€” operates only on files already in `state/demeter/library/`; no path traversal; no per-call argv
- **HTTP rate limits inherited from existing serve()** â€” no new throttle needed
